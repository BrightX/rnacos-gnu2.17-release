rnacos release build for gnu2.17 x86_64
========================

rnacos release build for gnu2.17 x86_64 (x86_64-unknown-linux-gnu - based on CentOS 7).

> For EOL Linux distribution, CentOS 7, Ubuntu 14 ~ 18, etc.

## R-NACOS

> **Fast Nacos**
>
> Nacos service platform implemented by Rust language, compatible with Nacos(Java)
